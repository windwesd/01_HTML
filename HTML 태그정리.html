글자관련태그

    1. 띄어쓰기 : & nbsp ; (세미콜론도 붙여 써야됨!!)
    2. 엔터 : br(Line BReak) 태그
    3. p(Paragraph : 문단) 태그
       글을 작성하는 용도의 태그	
       pre태그 처럼 형식이 유지되진 않음
       다른 문단, 태그 사이에 여백이 생성됨
    4. b(bold) 태그 : 단순히 글자를 굵게 표시하는 태그
    5. i(italic) 태그 : 단순히 글자를 기울여 표시하는 태그
    6. strong 태그 : 글자(단어)를 강조하는 용도의 태그
    7. em 태그 : 글자(문장)를 강조하는 용도의 태그
    8. mark 태그 :  형광펜 효과 를 추가하는 태그
    9. u 태그 : 밑줄을 긋는 태그
    10. s 태그 : 글자 가운데 선을 긋는 태그(취소선, 잘못 작성됨)
    11. del 태그 : 글자 가운데 선을 긋는 태그(삭제선, 이제 없음을 표시)
    12. small 태그 : 글자를 작게 표시하는 태그
    13. sup 태그 (윗첨자) : x3
    14. sub 태그 : (아랫첨자) : log25
    15. addr(abbreviation : 약어) 태그 : 마우스 오버 시 툴팁 표시 HTML

목록관련태그

    1. ul(Unordered List) 태그 : 순서가 없는 목록
    2. ol(ordered List) 태그 : 순서가 있는 목록
    3. li(List Item) 태그 : 목록의 한 줄
    4. type 속성 : 표시되는 숫자 형식 지정
       type="a" : 영어 소문자(a,b,c,d,...)
       type="A" : 영어 대분자(A,B,C,D,...)
       type="i" : 로마 소문자(i,ii, ... v, ... x) 
       type="I" : 로마 대문자(I,II, ... V ... X)
       start 속성 : 목록 시작 번호
       reversed : 순서 뒤집기(역순, 시작부터 -1씩)
